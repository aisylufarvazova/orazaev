Задача 1
Работая над алгоритмом быстрого вычисления a^n студент ошибся и написал код:
unsigned long f (unsigned int a, unsigned int n) {
    unsigned long r = 1;
    while (n > 0) {
        if (n % 2 == 1) {
            r *= a;
            n--;
        } else {
            r *= r;
            n /= 2;
        }
    }
    return r;
}
Что будет вычислять функция? Когда ответ будет равен а^n?

##########
Решение:
Функция точно не будет вычислять a^n, так как в случае, когда
функция вызывается с четным вторым аргументом, резльтат
умножается на себя(на 1) и мы теряем половину степеней.

Что эта функция вычисляет? Тут ответ несколько глупый, но
однозначный: то, что написал студент в коде.

А a^n функция будет равняться когда n = 0, 1, или
простому числу.
